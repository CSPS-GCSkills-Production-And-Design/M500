Préparer l’ensemble de la demande de soumissions PR

Gérer le processus de demande de soumissions (le responsable technique doit répondre aux questions techniques) C PR
Évaluer les éléments techniques des soumissions PR C

Négocier un contrat, le cas échéant C PR
Obtenir l’approbation du contrat C PR
Préparer et émettre des contrats PR

Surveiller le travail de l’entrepreneur, recevoir les livrables relatifs au contrat PR C
Surveiller les mouvements de trésorerie PR C
Signaler tout problème à l’agent de négociation des marchés PR
Résoudre tout problème relatif au contrat C PR
Surveiller la conformité aux modalités du contrat P P
Déterminer si les biens reçus et les services exécutés sont conformes aux besoins PR C

Confirmer qu’une modification au contrat est le moyen approprié PR
Négocier la modification apportée au contrat C PR
Obtenir l’approbation de la modification ou de l’ordre de modification PR
Préparer et émettre la modification au contrat PR

Vérifier la réception et l’approbation de tous les livrables P P
Retourner les équipements, le matériel et tout document fournis par le gouvernement P P
Transférer la propriété intellectuelle achevée (pour les contrats comprenant des travaux pour lesquels l’État reprend la possession de la propriété intellectuelle) P PR
Achever l’aliénation des biens appartenant à l’État PR
Émettre la dernière modification au contrat (si les fonds alloués n’ont pas été entièrement dépensés) PR
